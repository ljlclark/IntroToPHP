PHP Variables

PHP variables start with the "$" dollar sign character.

$text = "This is some text.";
The data type for $text becomes "string" because a string value was assigned.

Later in the code the type can by changed by assigning a different type value to the existing variable.
$text = 5; // The data type changes to "integer".

Changing the data type of an existing variable can cause confusion and lead to program errors that are hard to find.

The strict declaration causes an attempted change of data type to display an error. 

    <?php
      declare(strict_types=1);
      // PHPFile.php

      $text = "Some text.";
      $text = 5;  // This will display an error.
    ?>
