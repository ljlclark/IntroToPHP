The Windows Command Prompt

The Windows Command Prompt is a window where system commands can be entered.

This is the quickest and easiest way to display a command prompt in a specific folder.

1. Display the File Explorer by clicking the right mouse button on the Windows Icon at the bottom left of the screen and selecting the "File Explorer" menu option.

2. In the File Explorer, navigate to the folder where the commands are to be performed.

3. In the Navigation bar, click the left mouse button on the Folder icon just to the left of the displayed current folder path. This will highlight the current folder path.

4. Type "cmd" and press the ENTER key.
