Creating the TestForm.html File

Create the file TestForm.html in your program folder. See Create Folder or File.

Copy the main HTML code below and "paste" into the TestForm.html file. See Copy and Paste. 

    <!DOCTYPE html>
    <!-- TestForm.html -->
    <html lang="en">
    <head>
      <title>Test Javascript Form</title>
      <meta name="author" content="Lester J. Clark" />
      <meta charset="utf-8" />
      <meta name="viewport" content="width=device-width initial-scale=1" />
      <link rel="stylesheet" type="text/css" href="Intro.css" />
    </head>
    <body>
      <div class="page">
        <h3 id="pageTitle">Testing a Javascript Form</h3>

        <!-- The HTML Form will go here. -->

      </div>
    </body>
    </html>
