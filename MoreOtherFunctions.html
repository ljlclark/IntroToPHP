Other Folder or File Functions

Open the Windows File Explorer. see File Explorer.

Display a local HTML file in the Browser

To Display an HTML file; just double click the left mouse button on a file with the extension ".html". This type of file is usually associated with a browser that will open automatically to display the selected HTML file.

Rename Folder or File

To change the name of a folder or file in the right window pane; click the left mouse button on the name text to display the edit box. The first click may only select the file name so click the text again to display the edit box and type the changes. Use the left and right keys to move the insert cursor. Use the "delete" key to delete characters.

Open File for Edit

Click the right mouse button on the icon of the text file to be edited. This will display a popup menu.

Hover the mouse cursor over the "Open with" menu option. This will expand the menu options.

Frequently used editors are Notepad and Microsoft Visual Studio 2022. Click the left mouse button on an editor. The selected editor will open with the selected file.

Run a Command File

Double click the left mouse button on the name of the command file to to run that command file.
