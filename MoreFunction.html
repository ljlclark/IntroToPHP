PHP Functions

A PHP function is a group of code that can be called by name.

A PHP function can have values passed to it. In the function definition these values are called parameters. The function parameters are located between the function parentheses.

Function parameters can have a required data type.

A function can also have a required return type. The return type is separated from the function parameters by a colon. 

    <?php
    declare(strict_types=1);

    // Encode to HTML entities.
    function Display(string $text) : string
    {
      $retValue = htmlspecialchars($text);
      return $retValue;
    }
    ?>
