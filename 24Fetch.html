Calling the Fetch() Method

Create the file TestFetch.html. See Create Folder or File.
Copy and Paste the following HTML into the TestFetch.html file. See Copy and Paste. 

    <!DOCUMENT html>
    <html>
    <body>
      Click the left mouse button on the page to update with the WebAPI data.
      <div id="value"></div>
      <script>
        document.addEventListener("click", GetData);

        // Get the data without refreshing the page.
        function GetData()
        {
          fetch("http://localhost/ATestForm/TestFetch.php")
            .then((response) => response.text())
            .then((data) => HandleData(data));
            //.then((data) => { value.innerHTML = data; });
        }

        // Handle the retrieved data.
        function HandleData(data)
        {
          value.innerHTML = data;
        }
      </script>
    </body>
    </html;
