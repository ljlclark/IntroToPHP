Calling Static Class Functions

The syntax: ClassName::FunctionName() is used if a static function is called from outside the class.

The syntax: self::FunctionName() is used if a static function is called from another function in the Class. 

    LJCText::Line("Output this line.");

    // Contains common static functions.
    class LJCText
    {
      public static function Line(?string $text = null, ?bool $display = false
        , ?bool $lastPre = false)
      {
        if ($display)
        {
          $text = self::Display($text);
        }
        ... (Indicates there is more text that has been left out)
      }
      ...
    }
