External Code

Create the file LJCTextLib.php in the "PHPCode" program folder and copy the LJCText class to this external file. This allows it to be used by other PHP files.

It is useful to name the file with "Lib" to indicate that it is a collection of reusable code. This also prevents a conflict if the main internal class already has the same name. 

    <?php
      // LJCTextLib.php
      class LJCText
      {
        public static function Display(...
        public static function Line(...
      }
    ?>

The "require_once" line is included in the file that wants to use the externally defined classes. 

    <?php
      // Example.php
      // Full path is used to prevent difficulties with relative paths.
      $devPath = "c:/Users/Les/Documents/Visual Studio 2022/WebSitesDev/HTMLTutor";
      require_once "$devPath/IntroToPHP/PHPCode/LJCTextLib.php";

      LJCText::Line("This is an output line.");
    ?>
