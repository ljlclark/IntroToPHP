External Code

Create the file LJCHTMLLib.php in the "PHPCode" program folder and copy the LJCHTML class to this external file. This allows it to be used by other PHP files.

It is useful to name the file with "Lib" to indicate that it is a collection of reusable code. This also prevents a conflict if the main internal class already has the same name. 

    <?php
      // LJCHTMLLib.php
      class LJCHTML
      {
        public static function Encode(...
        public static function Line(...
      }
    ?>

The "require_once" line is included in the file that wants to use the externally defined classes. 

    <?php
      // Example.php
      // Full path is used to prevent difficulties with relative paths.
      $rootPath = "c:/Users/Les/Documents/Visual Studio 2022";
      $devPath = "$rootPath/WebSitesDev/HTMLTutor/IntroToPHP";
    require_once "$devPath/PHPCode/LJCHTMLLib.php";

      LJCHTML::Line("This is an output line.");
    ?>
