The Program to Use the Form Data

Create the program that will use the submitted form data. 

    <?php
      // TestForm.php
      declare(strict_types=1);
      require_once "LJCHTMLLib.php";

      // Get the Form data by name.
      $firstName = LJCHTML::Scrub($_POST["firstName"]);
      $lastName = LJCHTML::Scrub($_POST["lastName"]);

      // Create the returned HTML.
      LJCHTML::Begin("TestFormOutput.html", "Test Form Output");
      LJCHTML::BodyBegin();
      echo("Hi $firstName, $lastName");
      LJCHTML::End();
    ?>
