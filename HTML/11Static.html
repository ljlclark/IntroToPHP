Static Class Functions

The previous custom functions Encode() and Line() were created in the PHP file. They are Global functions.

If another PHP file is "included" and it also has a Global Line() function; that function will replace the previous function.

The Global functions can be placed inside a Class. This will protect the functions from being replaced. Notice that the Class could also be replaced. That is why Classes should be given names that attempt to be unique. 

    // Contains common static functions.
    class LJCHTML
    {
      // Encode to HTML entities to display HTML.
      public static function Encode(string $text) : string
      {
        $retValue = htmlspecialchars($text);
        return $retValue;
      }

    }
