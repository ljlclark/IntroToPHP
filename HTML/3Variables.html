PHP Variables

PHP variables start with the "$" dollar sign character.

$text = "This is some text.";
The data type for $text becomes "string" because a string value was assigned.

Later in the code the data type can by changed by assigning a different type value to the existing variable.
$text = 5; // The data type changes to "integer". 

    <?php
      // PHPFile.php

      // string
      $text = "Some text.";

      // int
      $text = 5;

      // bool
      $text = true;

      // float
      $text = 2.0;
    ?>
