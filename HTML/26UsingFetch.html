Using the WebAPI with Fetch

Create the folder: C:\inetpub\wwwroot\ATestForm if it does not already exist.

Copy LJCHTMLLib.php, TestFetch.html and TestFetch.php to the ATestForm folder.

Open the browser and type the URL "LocalHost/ATestForm/TestFetch.html". Then press the ENTER key.

The HTML Form will be displayed. Click the left mouse button on the form.

The retrieved text "The retrieved value" will appear on the form.

If there are problems, check the spelling and character case of the code.

Also check the web server configuration.
