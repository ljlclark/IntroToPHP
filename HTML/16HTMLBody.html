Other HTML Output Methods

Add these methods to the LJCHTML Class. See the External Page 

    // Writes the "head" end and "body" begin elements.
    public static function BodyBegin()
    {
      self::Line("</head>");
      self::Line("<body>");
    }

    // Writes the HTML ending lines.
    public static function End()
    {
      self::Line("</body>");
      self::Line("</html>");
    }
