Creating a Folder or File

Open the Windows File Explorer.

Create Folder

To create a Folder, click the right mouse button on an empty space in the right pane that is not highlighted. This will display a popup menu.

Select the "New" menu item. This will expand the menu. Select the "Folder" menu option. Type the name of the folder in the edit box provided for the new folder name.

Create File

To create a new File, display the popup menu as above and select the "Text Document" menu option. Type the name of the file in the edit box provided for the new file name.
