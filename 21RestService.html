Retrieving Data with REST

REST stands for "Representational State Transfer".

A PHP program that receives request values in the URL is acting similar to a REST Web Service. It can return HTML or other types of data. 

    <?php
      // TestRest.php
      declare(strict_types=1);
      require_once "LJCTextLib.php";

      $name = LJCText::Scrub($_GET["name"]);
      echo("name: $name");
    ?>
