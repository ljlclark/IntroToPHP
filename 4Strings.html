The PHP "string" Data Type

The dot "." operator is the string concatenation operator. It combines two strings into one.

$text = "This is " . "some text.";
// $text = "This is some text";

The concatenation assignment operator is ".=". It is a short cut for $text1 = $text1 . $text2;. This can be coded as $text1 .= $text2;.

$text = "This is ";
$text .= "some text.";
// $text = "This is some text";
