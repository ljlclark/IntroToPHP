Retrieving Data with REST

REST stands for "Representational State Transfer".

A PHP program that receives request values in the URL is acting as a REST Web Service. It can return HTML or other types of data. 

    <?php
      // TestRest.php
      declare(strict_types=1);
      require_once "LJCHTMLLib.php";

      // Get values from the URL.
      $name = LJCHTML::Scrub($_GET["name"]);

      // Return the data.
      echo("name: $name");
    ?>
