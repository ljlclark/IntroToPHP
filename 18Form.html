Creating an HTML Form

An HTML form collects information to pass to a web server program.

The program to receive the information is specified in the "action" attribute of the "form" element.

Copy the Form HTML code below and "paste" it into the TestForm.html file at the comment "<!-- The HTML Form will go here. -->". See Copy and Paste. 

    <form action="TestForm.php" method="post">
      <label for="firstName">First Name:</label>
      <input type="text" name="firstName" /><br />
      <br />
      <label for="lastName">Last Name:</label>
      <input type="text" name="lastName" required /><br />
      <br />
      <input type="submit" value="Submit" />
    </form>

This form gets the First Name and Last Name from the user.
