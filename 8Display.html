An Encode() PHP Function

HTML entities must be used to display HTML characters on a page. 

    echo("&lt;html lang=&quot;en&quot;&gt;\r\n");

It is inconvenient and confusing to type the HTML Entities each time they are needed.

A PHP function can be created to convert the HTML characters to HTML entities. 

    // Encode to HTML entities.
    function Encode(string $text) : string
    {
      $retValue = htmlspecialchars($text);
      return $retValue;
    }

Now the statements can be written like this: 

    echo(Encode("<html lang=\"en\" html>\r\n"))

There are several ways to encode HTML entities instead of using the htmlspecialchars() function. By putting this in the Encode() function instead of calling it directly, it can be changed later without having to change every place it is used.
