A Line() PHP Function

It is inconvenient and distracting to type the "\r\n" characters after every line, especially if there are many lines to be output. 

    echo("<!DOCTYPE html>\r\n")
    echo("<html lang=\"en\">\r\n")
    echo("</html>\r\n")

The Line() function can output the text and add those characters. 

    // Writes the supplied text to the output stream with the new line values.
    function Line(string $text)
    {
      echo($text . "\r\n");
    }

The output is easier to type and read. 

    Line("<!DOCTYPE html>")
    Line("<html lang=\"en\">")
    Line("</html>")
