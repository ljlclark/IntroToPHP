The "echo" Statement

The "echo" statement writes to the output stream. It can be written as echo "Some Text"; or echo("SomeText");

The output stream is the text that will be sent to the client browser. The output text for the browser is usually HTML.

Create the file TestEcho.php. See Create Folder or File.
Copy and Paste the following PHP into the TestEcho.php file. See Copy and Paste. 

    <?php
      echo("<!DOCTYPE html>\r\n");
      echo("<html lang=\"en\">\r\n");
      echo("<body>\r\n");
      echo("This is some test HTML generated by TestEcho.php");
      echo("</body>\r\n");
      echo("</html>\r\n");
    ?>

Notice the escape characters "\r\n". These are the escape characters for a "return" and "new line". This is included so that if the output is sent to a file, it will be easy to read. These characters are ignored by the browser. Also notice the escape character "\"". This is required to output a double quote.
