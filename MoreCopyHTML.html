How to Copy and Paste

These are the steps To copy and past text from an HTML page or most Windows editors.

Copy

On the HTML page, select the text to be copied by holding down the left mouse button over the beginning of the section to be copied and dragging the mouse to the end of the text to be copied. This will highlight the text.

In most Windows editors the same thing can be done by moving the cursor to the beginning of the text, holding down the "Shift" key and using the Left, Right, Up and Down keys.

On an HTML page, when the left mouse button is released, a popup menu icon will appear with three dots. Move the mouse pointer over the icon and the popup menu will appear. Select "Copy" from the menu. This will copy the text to the clipboard.

You can usually copy highlighted text to the clipboard by holding down the Control (Ctrl) key and typing the "C" key. "Ctrl-C".

Usually in other editors, click the right mouse button to display the context menu and select "Copy".

Paste

Open the file where the text will be pasted. Move the cursor to the place where the text is to be inserted.

Click the right mouse button to display a context menu and select "Paste".

You can usually paste text from the clipboard by holding down the Control (Ctrl) key and typing the "V" key. "Ctrl-V".
