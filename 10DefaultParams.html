Default Function Parameters

This is an updated version of the Line() function that uses default parameters. The question mark "?" in front of a parameter data type means that the parameter is allowed to be null or in other words is nullable. 

    function Line(?string $text = null, ?bool $encode = false
      , ?bool $lastPre = false)
    {
      if ($encode)
      {
        $text = Encode($text);
      }
      if ($lastPre)
      {
        $text = $text . "</pre>");
      }
      echo($text . "\r\n");
    }

Default parameters can be left blank. If a default parameter is blank, all parameters after that must be blank or specified with a named parameter. 

    Line(<pre>);
    Line();
    Line("<!DOCTYPE html>", true);
    Line("<html lang="en">", true);
    Line("A line of text.", lastPre: true);
