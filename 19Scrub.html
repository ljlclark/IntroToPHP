The Scrub Method

Any data received by a server program should be cleaned to prevent harm to the program, data or computer. This process is sometimes called scrubbing.

Add the Scrub function to the LJCHTML class in LJCHTMLLib.php. 

    // Returns a scrubbed external value.
    public static function Scrub(string $text) : string
    {
      $retValue = trim($text);
      $retValue = stripslashes($retValue);
      $retValue = htmlspecialchars($retValue);
      return $retValue;
    }
