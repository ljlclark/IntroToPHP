Entering the Data

Create the folder: C:\inetpub\wwwroot\ATestForm.

Copy LJCTextLib.php, TestForm.html and TestForm.php to the new folder.

Open the browser and type the URL "LocalHost/ATestForm/TestForm.html". Then press the ENTER key.

The Javascript Form will be displayed. Enter the data and Press the "Submit" button. The Submit will also be performed from any field in the form if the ENTER key is pressed.

The program that uses the form data will display: "Hi" and the entered first and last name.

If there are problems, check the spelling and character case of the code.

Also check the web server configuration.
