Entering the Form Data

Create the folder: C:\inetpub\wwwroot\ATestForm if it does not already exist.

Copy LJCTextLib.php, TestForm.html and TestForm.php to the new folder. See Folder or File Actions.

Open the browser and type the URL "LocalHost/ATestForm/TestForm.html". Then press the ENTER key.

The HTML Form will be displayed. Enter the data and Press the "Submit" button. The Submit will also be performed from any field in the form if the ENTER key is pressed.

The program that uses the form data will display: "Hi" and the entered first and last name.

A PHP program that receives form data and returns HTML is acting as a Web Application usually referred to as a Web App.

If there are problems, check the spelling and character case of the code.

Also check the web server configuration.
