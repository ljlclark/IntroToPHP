The TestEcho Command File

Create the file TestEcho.cmd. See Create Folder or File.

Edit the file TestEcho.cmd to contain the command "PHP TestEcho.php > TestEcho.html".
Or use the windows command: echo PHP TestEcho.php ^> TestEcho.html > TestEcho.cmd See The Windows Command Prompt.

Run the command file to create the output HTML. See Other Folder or File Actions.

Display the HTML file in the browser to see the output.
