A Very Simple WebAPI

A WebAPI is a Web Application Programming Interface. It is used to retrieve program or database data.

A WebAPI is often used in HTML to retrieve data and modify the Web Page without having to refresh the page from the server.

The Javascript code used to retrieve data is the XMLHttpRequest object.

A newer Javascript method is Fetch(). This is an easier method to use and is the one demonstrated on the next page.

First create the very simple PHP WebAPI program below. It just returns some data but that data could have come from another program or database. See Folder or File Actions 

    <?php
      // TestForm.php
      $text =  "value";
      echo("the retrieved $text");
    ?>
