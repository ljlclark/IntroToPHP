Using the REST Web Service

Create the folder: C:\inetpub\wwwroot\ATestForm if it does not already exist.

Copy LJCTextLib.php and TestRest.php to the ATestForm folder. See Folder or File Actions.

Open the browser and type the URL "LocalHost/ATestForm/TestRest.html?name=John". Then press the ENTER key.

The program that uses the REST data will display: "name: John".

If there are problems, check the spelling and character case of the code.

Also check the web server configuration.
