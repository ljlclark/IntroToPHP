HTMLBegin Output Method

Typing the lines for the beginning of an HTML document every time they are needed or copying and pasting them from another file can be tedius.

The HTMLBegin() method can be added to the LJCText Class to make it easier. See the External Page 

    class LJCText
    {
      // Writes the HTML beginning lines.
      public static function HTMLBegin(string $outputFileName, string $pageTitle)
      {
        self::Line("<!DOCTYPE html>");
        self::Line("<!-- $outputFileName -->");
        self::Line("<html lang=\"en\">");
        self::Line("<head>");
        self::Line("  <title>$pageTitle</title>");
        self::Line("  <meta name=\"author\" content=\"Lester J. Clark\" />");
        self::Line("  <meta charset=\"utf-8\" />");
        echo("  <meta name=\"viewport\" content=\"width =device-width");
        self::Line(" initial-scale=1\" />");
      }
    }

Notice how variables can be include in a string. They will be replaced with the variable value.
